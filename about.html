About the Project

This project is a Library Management System developed as part of the Master of Computer Applications course at Poddar Management and Technical Campus. It is designed to streamline library operations such as book management, issuing/returning books, and user tracking.

The project was completed under the guidance of Prof. Ruchir Saxena.

Developed by: Aryan Arora and Chitransh Mathur.

This system is intended for academic and educational use only.

Complete project source code and documentation can be found on GitHub.
